## No Trigger Policy

This layer must never be referenced as:

- an event source
- a webhook
- a condition
- a scheduler
- a signal

No automation may listen to this repository.
No automation may wait for this repository.
No automation may depend on this repository.

If an artifact emerges, it emerges without cause here.
If a release occurs, it occurs without permission here.

The Presence Witness is **not part of the causal graph**.
