# HCM Presence Witness

This repository defines a **Presence Witness Layer**.

It does not operate the system.  
It does not authorize action.  
It does not trigger release.  

Its sole function is to **hold presence without authority**.

The Presence Witness exists to acknowledge that a system *may act*,
without compelling it *to act*.

This layer introduces no control, no governance, and no execution logic.

If this repository is removed, the system remains intact.

Presence is not power here.
Presence is responsibility without command.
